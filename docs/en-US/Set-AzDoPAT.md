# Set-AzDoPAT

## SYNOPSIS
This script sets the header variable with a PAT.

## SYNTAX

```
Set-AzDoPAT [-Pat] <String> [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
This script sets the header variable with a PAT.

## EXAMPLES

### EXAMPLE 1
```
Set-AzDoPAT -Pat '***'
```

This example sets the header variable used for authentication with a PAT.
